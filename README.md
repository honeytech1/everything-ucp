## üöÄ What is UCP?

**Universal Commerce Protocol (UCP)** is an **open-source protocol** that standardizes how AI agents, merchants, platforms, and payment systems interact across the full commerce lifecycle ‚Äî from product discovery to post-purchase support.

### Key Features

- **Transport Agnostic**: Works via REST APIs, MCP, A2A, or Agent2Agent protocols
- **Composable Architecture**: Modular capabilities (Checkout, Orders, Catalog) with independent versioning
- **Merchant-First**: Retailers remain Merchant of Record with full customer data ownership
- **Payment Flexible**: Compatible with AP2 for secure agentic payments with cryptographic consent
- **Open & Extensible**: Community-driven with Apache 2.0 license

### What Problem Does UCP Solve?

**The N√óN Integration Problem**

Without UCP:
- ‚ùå Custom integration for ChatGPT
- ‚ùå Custom integration for Google Gemini  
- ‚ùå Custom integration for Microsoft Copilot
- ‚ùå Custom integration for every AI platform
- ‚ùå Unsustainable complexity

With UCP:
- ‚úÖ Single protocol implementation
- ‚úÖ Works across all UCP-enabled AI platforms
- ‚úÖ Standardized discovery, checkout, and fulfillment
- ‚úÖ Future AI platforms automatically compatible

## üîó Related Protocols

### Compatible Protocols

#### Model Context Protocol (MCP)
**Website:** [modelcontextprotocol.io](https://modelcontextprotocol.io/)

**What it does:** Enables AI model interactions and context transport  
**UCP Integration:** Built-in MCP transport support  
**Use case:** AI agents communicate with UCP merchants via MCP

```
AI Agent ‚Üí MCP Transport ‚Üí UCP Merchant Server
```

**Repository:** [GitHub - MCP TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk)

#### Agent Payments Protocol (AP2)
**Website:** [developers.google.com/wallet/agent-payments](https://developers.google.com/wallet/agent-payments)

**What it does:** Secure payment protocol with cryptographic consent  
**UCP Integration:** Fully compatible, recommended for payments  
**Key Features:**
- Cryptographic proof of user authorization
- Payment mandate support
- Verifiable credentials
- Open wallet ecosystem

#### Agent2Agent (A2A)
**Website:** [agent2agent.dev](https://agent2agent.dev/)

**What it does:** Protocol for agent-to-agent communication  
**UCP Integration:** Supported transport method  
**Use case:** Direct agent-to-merchant messaging

### Alternative Protocols

#### Agent Commerce Protocol (ACP)
**Website:** [agentic-commerce-protocol.org](https://agentic-commerce-protocol.org/)

**Developed by:** OpenAI with Stripe  
**Status:** Alternative standard  
**Focus:** Tighter ChatGPT integration, Stripe-centric payments

**UCP vs. ACP Comparison:**

| Aspect | UCP | ACP |
|--|--|--|
| **Philosophy** | Broad coalition, vendor-neutral | ChatGPT-first, faster to market |
| **Co-Developers** | Google, Shopify, Etsy, Wayfair, Target, Walmart | OpenAI, Stripe |
| **Payments** | Flexible payment handlers | Stripe-centric |
| **Adoption** | 20+ partners across ecosystem | OpenAI/Stripe partnership |
| **Integration** | REST, MCP, A2A support | Optimized for ChatGPT |
| **Open Source** | Apache 2.0, fully open | Partially open |

**Future:** Protocols may interoperate as standards mature. Some retailers may support both.

### For Developers

#### Quick Start Tutorial

```python
from ucp_sdk import Checkout, LineItem

# Create checkout session
def create_checkout(items, buyer_email):
    checkout = Checkout(
        currency="USD",
        buyer={"email": buyer_email},
        line_items=[
            LineItem(
                id=item["id"],
                title=item["name"],
                quantity=item["qty"],
                price=item["price_cents"]
            )
            for item in items
        ]
    )
    return checkout
```
